About Poke Compendium

Poke Compendium is a powerful tool designed for Pokémon trainers to access all the essential information about their favorite Pokémon and items, brought to you by the data provided through the PokeAPI.

Project Overview

This project is built with Angular and consumes data from the PokeAPI. It offers a user-friendly interface to search for Pokémon, items, and other related content.

Technology Stack

The project utilizes the following technologies:

  • Angular
  • TypeScript
  • PokeAPI (RESTful API)
